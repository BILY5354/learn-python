# 小技巧
- 字符串格式化
  - `r` 不使用转义
  - `f` 格式化 让`{变量}`
- 列表 `+` 与 `append`
  - `+` 是将两个列表合并
  - `append` 是加入 比如列表
  - 例子可以看练习 文件目录
